# about project

## app's purpose
"To do list" app
to practice React, Material UI,Typescript, SCSS.

## major functions
・write a memo in the form and add (plus button)
・modify each memo (pen icon)
・delete each memo (bin icon)
・delete all memos (Delete all button)
